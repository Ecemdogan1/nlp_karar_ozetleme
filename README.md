# nlp_karar_ozetleme
Bu proje, uzun mahkeme kararlarını doğal dil işleme (NLP) teknikleri kullanarak analiz eden ve karar metinlerini Olay, Hukuki Sorun, Mahkeme Kararı ve Sonuç başlıkları altında özetleyen bir sistemdir. Proje kapsamında PDF işleme, metin parçalama (chunking), özetleme (summarization) ve prompt engineering teknikleri uygulanmıştır.
